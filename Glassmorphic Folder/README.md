# Glassmorphic Folder Explorer

A stunning, premium glassmorphic folder explorer built with React, Vite, and Tailwind CSS.

## Features
- **Modern Glassmorphic Design**: High-end visuals using backdrop-blur and subtle gradients.
- **Micro-animations**: Smooth transitions and hover effects for a responsive feel.
- **Interactive Stacks**: Folder content fanned out on hover.
- **Full View Mode**: Detailed view for folder contents.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Folder Structure
- `src/App.jsx`: Main interface and application shell.
- `src/components/`: Reusable components.
- `src/index.css`: Tailwind CSS and global styles.
- `src/main.jsx`: React entry point.
- `src/components/FolderPrototype.jsx`: An alternative folder prototype with advanced configuration settings.
